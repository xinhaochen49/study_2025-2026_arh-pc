# Лабораторная работа №2. Система контроля версий Git

## Цель работы
Целью работы является изучение идеологии и применения средств контроля версий,
приобретение практических навыков по работе с системой контроля версий git.

## Выполнение работы
1. Настройка GitHub и SSH-ключей
2. Настройка git
3. Создание рабочего пространства
4. Клонирование репозитория

## Результаты
- Настроен SSH-ключ для GitHub
- Настроены параметры git
- Создана структура каталогов курса
- Клонирован репозиторий из шаблона

## Выводы
В ходе работы были освоены базовые навыки работы с системой контроля версий Git.
